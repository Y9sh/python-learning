# Python Learning Journey

This repository contains basic Python exercises, experiments, and small projects created during my self-learning journey.

The projects here were built to practice problem solving, programming logic, and understanding Python fundamentals through hands-on experimentation.

## Learning Approach

Some projects include AI-assisted code or commented examples used as part of the learning process. I use AI to:
- compare solutions
- identify mistakes
- understand different coding approaches
- explore possible optimizations

The goal of this repository is learning, experimentation, and gradual improvement in software development and backend-related programming.
